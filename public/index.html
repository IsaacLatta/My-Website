This Web Server

Technical Description

The web content you are seeing right now is currently being served from my own web server.

This website is running on my personal server, I have constructed an xml file in which my server reads its configuration from, and python scripts to handle the some of the actionable components of this website (such as contacting me). This website is hosted in either the cloud or on some of my old hardware, depending on when you read this.

The website is running through a series of proxy servers. First through a WAF(Web Application Firewall). Then it is proxied on the machine to a series of web server processes. Depending on the time of reading this, it is proxied by either nginx, or my own load balancer. I have also configured a firewall and EDR software to harden the server. Additionally I have implented server side scripts for managing the proxies and docker images these server processes run in.

This web server is written in C++ using ASIO for networking, OpenSSL for HTTPS, and jwt-cpp for authentication.
